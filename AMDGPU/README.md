# Ja-JP Environment setup Meta Package

Installs AMD GPU related drivers.

## Install

* `yay -Bi .` to generate the Package and install
* use the `install-pkgs.sh` script in the project root if `yay -Bi` fails

### Other settings

* [Video Acceleration](wiki.archlinux.org/title/Hardware_video_acceleration#AMD/ATI)
  * Check status using `vainfo`
